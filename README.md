Скоринг кандидатов по резюме

Проект включает в себя следующие этапы реализиции:

- разработка промпта для анализ резюме кандидата на предмет соответствия вакансии, написание кода на базе Playground GPT (Open AI)
- создание интерфейса с помощью Python-библиотеки Streamlit (в окна вводятся описание вакансии и резюме, по кнопке получаем анализ кандидата)
- доработка кода, позволяющего вводить в окна веб-приложения ссылки на вакансии и резюме
- развертывание проекта через Streamlit Community Cloud
