Скоринг кандидатов по резюме.

Проект включает в себя следующие этапы реализиции: 
- на базе Playground GPT (Open AI) разработка промпта для анализ резюме кандидата на предмет соответствия вакансии, написание кода по автоматизации процесса скоринга.
- создание интерфейса с помощью Python-библиотеки Streamlit (в окна вводятся описание вакансии и резюме, по кнопке получаем анализ кандидата).
- доработка кода, осуществлющего парсинг сайта с вакансиями (в окна веб-приложения вводятся ссылки на вакансии и резюме).
- развертывание проекта через Streamlit Community Cloud.
